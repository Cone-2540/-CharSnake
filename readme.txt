游戏名：CharSnake 汉字贪吃蛇

设计思路：相比于图形化，我受到steam上《文字游戏》的启发，利用汉字的正方形特点，将游戏界面全部改成了汉字。

游玩方法：打开output中的snake.exe文件或者使用vscode运行snake.cpp

操作方法：主界面有ws操作上下移动，回车确认，其他具体操作可见主界面的规则说明
其他地方也有按键提示


实现的功能：
除了图形化基本都实现了

可能的BUG以及修复方法：
①过于频繁存档可能导致炸档。这时需要手动清除output里面save.txt的内容。
②游戏暂不支持清除历史记录，如果想清除可以手动修改output里面history_score.txt的内容

备注：
①由于一开始的编程是全英文的，中途才改变思路，因此早期代码的注释用UTF8编码的都变成了乱码，后期的没有问题。
②虽然思路几乎是由我个人完成，但是许多代码工作还是不少copilot和Deepseek参与的。因此可能有些地方有AI味道。
③程序仍在DEBUG阶段，目前没有明显BUG，可能后续还会上传新版本
